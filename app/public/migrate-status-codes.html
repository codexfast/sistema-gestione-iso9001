🔄 Migrazione Codici Status

Migration 007: Allineamento con response_options DB

⚠️ Attenzione

Questa utility migra i codici status obsoleti in IndexedDB ai nuovi valori conformi alla Migration 007.

Cosa fa questa migrazione:

• Converte 'compliant' → 'C' (Conforme)

• Converte 'non_compliant' → 'NC' (Non Conforme)

• Converte 'partial' → 'OSS' (Osservazione)

• Converte 'not_applicable' → 'NA' (Non Applicabile)

• Rimuove elementi dalla sync queue con status obsoleti

Mapping Conversione:

compliant → C
non_compliant → NC
partial → OSS
not_applicable → NA
